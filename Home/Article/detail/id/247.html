记录用到的Linux命令

查找文件

find / -name [name] 可以使用*去匹配任意字符

查看储存空间(也能查看到挂载点)

df -hl

Screen相关

screen cmd 在新窗口运行命令

ctrl+a+d     切换窗口

screen -ls    查看存在的窗口

screen -r pid 恢复窗口

screen -S yourname -> 新建一个叫yourname的session

screen -r yourname -> 回到yourname这个session

screen -d yourname -> 远程detach某个session

screen -d -r yourname -> 结束当前session并回到yourname这个session

查看程序执行时调用动态库的过程

ldd name

从服务器上下载文件(SCP命令)

scp username@servername:/path/filename /var/www/local_dir

eg:scp root@127.0.0.1:/home/1.txt /1.txt

上传本地文件到服务器

scp /path/filename username@servername:/path  

从服务器下载整个目录

scp -r username@servername:/var/www/remote_dir/（远程目录） /var/www/local_dir（本地目录）

上传目录到服务器

scp  -r local_dir username@servername:remote_dir

挂载U盘硬盘

首先你得保证你的U盘的格式是fat格式

fdisk -l

mount -t vfat /dev/sdb /mnt/usb （如果没有这个目录可以mkdir usb）

umount /dev/sdb

程序运行时间time ./app

or

#include /<sys/time.h/>

struct  timeval  start;

struct  timeval  end;

unsigned long timer;

gettimeofday(&start,NULL);

gettimeofday(&end,NULL);

timer = 1000000 * (end.tv_sec-start.tv_sec)+ end.tv_usec-start.tv_usec;

printf("read stream timer = %ld us\n",timer);

apt相关

安装和卸载

apt-get install *

apt-get remove *

同步源

apt-get update

更新软件

apt-get upgrade

搜索软件

apt-cache search *

查看依赖版本等

apt-cache show *

zip/unzip

解压 unzip file.zip

压缩 zip -r file.zip file/*

如果文件过多

find file/ -name '*' -print | zip 1.zip -@
